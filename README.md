## üõ† How to Contribute

2.  **Fork:** Click the "Fork" button to create your own copy of this repository.
3.  **Translate:** * Copy the `en.json` content.
    * Create a new file for your language (e.g., `ha.json` for Hausa).

4.  **Pull Request (PR):** Submit a PR back to our `main` branch. Our team will review the JSON formatting and technical accuracy before merging.
